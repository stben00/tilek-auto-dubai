# 🚗 Tilek Auto Dubai — Сайт + Админ-панель

Полный пакет: лендинг + админка для управления контентом.

## 📁 Состав

```
📂 ваша-папка/
├── 📄 index.html          ← Главный сайт (для клиентов)
├── 📄 admin.html          ← Админ-панель (для вас)
└── 📂 assets/
    └── 🖼️ logo.png        ← Логотип
```

## 🚀 Как запустить

### Локально на компьютере
Просто откройте `index.html` в браузере двойным кликом. Для админки — откройте `admin.html`.

### На хостинге
Загрузите все 3 файла (index.html, admin.html, папку assets) в корень хостинга через FTP. Сайт будет доступен по адресу `вашсайт.com`, а админка — по `вашсайт.com/admin.html`.

## 🔥 Быстрый запуск для продаж

1. В `index.html` в блоке `window.TILEK_PUBLIC_CONFIG` укажите реальный WhatsApp, Instagram, WhatsApp-группу и 2GIS. Это публичные данные для всех посетителей.
2. Откройте `admin.html`
3. Зайдите в **Тексты сайта → Контакты** и продублируйте те же контакты для локальной админки
4. Загрузите фото владельца в разделе **Владелец**
5. Откройте `index.html` и проверьте с телефона кнопку WhatsApp

Теперь форма заявки не просто сохраняет данные в браузере: она создаёт готовое сообщение для WhatsApp. Клиент отправляет его одним нажатием, и заявка сразу попадает в переписку.

## 🔐 Вход в админку

- **Логин:** `admin`
- **Пароль:** `tilek2025`

⚠️ **Обязательно поменяйте пароль** в разделе «Настройки» сразу после первого входа!

## 🔄 Как админка связана с сайтом

Сайт `index.html` и админка `admin.html` обмениваются данными через **localStorage браузера**:

1. **Заявки с формы** → автоматически попадают в админку
2. **Авто / отзывы / FAQ / видео** из админки → обновляются на сайте при открытии/перезагрузке в том же браузере
3. **Заявки с формы** → превращаются в готовое сообщение WhatsApp
4. **Контакты из админки** (WhatsApp, Instagram, WhatsApp-группа, 2GIS) → подставляются на сайте автоматически
5. **Просмотры сайта** → считаются автоматически

### ⚠️ Важно знать

Данные хранятся **в браузере**, а не на сервере. Это значит:
- ✅ Бесплатно, не нужен сервер
- ✅ Работает мгновенно
- ⚠️ Заявки видны только в том браузере, где открыт сайт
- ⚠️ При очистке кеша браузера данные пропадают (делайте бэкапы!)
- ✅ WhatsApp-дублирование помогает не терять заявки даже без сервера

### 🔥 Рекомендация для серьёзного бизнеса

Если у вас будет много трафика и заявок — стоит подключить настоящий сервер с базой данных. Структура данных в админке уже готова для этого. Бесплатные варианты:
- **Netlify Forms** — заявки через email (бесплатно до 100/мес)
- **Formspree** — простая интеграция формы
- **Render.com + Node.js** — полноценный backend
- **Telegram/CRM webhook** — быстрые уведомления менеджеру

Если решите — могу помочь с миграцией.

## 🆘 Что-то не работает?

- **Не вижу заявок** → Проверьте, что сайт и админка открыты в одном браузере
- **Забыл пароль** → Откройте админку, в консоли (F12) введите: `localStorage.removeItem('tilek_auto_data')` → перезагрузите, логин/пароль сбросятся на admin/tilek2025
- **Хочу всё сбросить** → Настройки → «Удалить все данные»
