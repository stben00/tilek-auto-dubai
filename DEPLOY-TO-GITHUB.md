## 📋 Пошаговая инструкция

### Шаг 1. Создайте репозиторий

1. Откройте → https://github.com/new
2. Repository name: **`tilek-auto-dubai`**
3. Описание (опционально): `Авто из Дубая под ключ`
4. Выберите **Public** (это важно — для бесплатного GitHub Pages)
5. ✅ Поставьте галку **Add a README file**
6. Нажмите **Create repository**

### Шаг 2. Загрузите файлы

1. На странице репозитория нажмите **Add file → Upload files**
2. Перетащите в окно **все файлы из ZIP-архива**:
   - `index.html`
   - `admin.html`
   - папку `assets/` целиком (с logo.png внутри)
3. Внизу нажмите **Commit changes**

Перед загрузкой обязательно откройте `index.html` и замените значения в `window.TILEK_PUBLIC_CONFIG`: WhatsApp, Instagram и при необходимости webhook. Это публичные контакты для всех посетителей сайта.

### Шаг 3. Включите GitHub Pages

1. В репозитории откройте **Settings** (вкладка наверху)
2. В левом меню — **Pages**
3. В **Source** выберите:
   - Branch: **main**
   - Folder: **/ (root)**
4. Нажмите **Save**

### Шаг 4. Получите ссылку

Через 1-2 минуты ваш сайт будет доступен по адресу:

```
https://ВАШ-USERNAME.github.io/tilek-auto-dubai/
```

Где `ВАШ-USERNAME` — ваш логин на GitHub.

**Админка:**
```
https://ВАШ-USERNAME.github.io/tilek-auto-dubai/admin.html
```

## 🎯 Особенности

- ✅ Бесплатно, навсегда
- ✅ HTTPS включён автоматически
- ✅ Можно подключить свой домен (Settings → Pages → Custom domain)
- ✅ Форма заявки создаёт готовое сообщение в WhatsApp — это основной быстрый канал продаж
- ⚠️ Заявки и данные админки хранятся в localStorage браузера — у каждого посетителя свои. Для продакшена с общей базой нужен сервер, Formspree/CRM webhook или другой backend

## 🌐 Подключение домена (опционально)

Если есть свой домен (например, tilek-auto.com):

1. В DNS добавьте CNAME-запись: `www → ВАШ-USERNAME.github.io`
2. В Settings → Pages → Custom domain: введите `www.tilek-auto.com`
3. Подождите 10-30 минут — сайт заработает на вашем домене

## 🆘 Если что-то не работает

- **Страница 404 после деплоя** → подождите 2-3 минуты, GitHub должен пересобрать
- **Не видно картинки** → проверьте что папка `assets` загружена и пути в верхнем регистре правильные
- **Админка пустая** → нужно сначала зайти на главную страницу, потом на /admin.html
- **WhatsApp открывается с нулями** → зайдите в `admin.html` → «Тексты сайта» → «Контакты» и укажите реальный номер без `+`
